About

What is this for?

This site has been created as a guide for Defence website owners to help build new sites within the Defence framework and brand.

The templates and advice provided tools that help content creators ensure that their sites are on brand, up-to-date from a technical standpoint, and comply with the legal accessibility and compliance requirements.

What this doesn’t cover

This documentation and style guide are not intended as a replacement for technical expertise. It is intended to guide for web professionals in the requirements and details of the defence web standards and does not cover the technical aspects of web development such HTML, CSS and JavaScript.

Who is this for?

The majority of this site is aimed at web designers, developers and producers who are required to provide websites for the Defence network.

It is strongly advised that only personell with experience in the web design and development space should be using this framework.

NOTE: This is not point and click.

Reason

The template system provides a standardised toolset for content creators which if used correctly will ensure a consistent Australia Department of Defence web presence.
